6.1.1 Joint Distributions and Independence

For three or more random variables, the joint PDF, joint PMF, and joint CDF are defined in a similar way to what we have already seen for the case of two random variables. Let X1X1, X2X2, ⋯⋯, XnXn be nn discrete random variables. The joint PMF of X1X1, X2X2, ⋯⋯, XnXn is defined as

PX1,X2,...,Xn(x1,x2,...,xn)=P(X1=x1,X2=x2,...,Xn=xn).PX1,X2,...,Xn(x1,x2,...,xn)=P(X1=x1,X2=x2,...,Xn=xn).
For nn jointly continuous random variables X1X1, X2X2, ⋯⋯, XnXn, the joint PDF is defined to be the function fX1X2...Xn(x1,x2,...,xn)fX1X2...Xn(x1,x2,...,xn) such that the probability of any set A⊂RnA⊂Rn is given by the integral of the PDF over the set AA. In particular, for a set A∈RnA∈Rn, we can write
P((X1,X2,⋯,Xn)∈A)=∫⋯∫A⋯∫fX1X2⋯Xn(x1,x2,⋯,xn)dx1dx2⋯dxn.P((X1,X2,⋯,Xn)∈A)=∫⋯∫A⋯∫fX1X2⋯Xn(x1,x2,⋯,xn)dx1dx2⋯dxn.
The marginal PDF of XiXi can be obtained by integrating all other XjXj's. For example,
fX1(x1)=∫∞−∞⋯∫∞−∞fX1X2...Xn(x1,x2,...,xn)dx2⋯dxn.fX1(x1)=∫−∞∞⋯∫−∞∞fX1X2...Xn(x1,x2,...,xn)dx2⋯dxn.
The joint CDF of nn random variables X1X1, X2X2,...,XnXn is defined as
FX1,X2,...,Xn(x1,x2,...,xn)=P(X1≤x1,X2≤x2,...,Xn≤xn).FX1,X2,...,Xn(x1,x2,...,xn)=P(X1≤x1,X2≤x2,...,Xn≤xn).

Independence: The idea of independence is exactly the same as what we have seen before. We restate it here in terms of the joint PMF, joint PDF, and joint CDF. Random variables X1X1, X2X2, ...... , XnXn are independent, if for all (x1,x2,...,xn)∈Rn(x1,x2,...,xn)∈Rn,
FX1,X2,...,Xn(x1,x2,...,xn)=FX1(x1)FX2(x2)⋯FXn(xn).FX1,X2,...,Xn(x1,x2,...,xn)=FX1(x1)FX2(x2)⋯FXn(xn).
Equivalently, if X1X1, X2X2, ..., XnXn are discrete, then they are independent if for all (x1,x2,...,xn)∈Rn(x1,x2,...,xn)∈Rn, we have
PX1,X2,...,Xn(x1,x2,...,xn)=PX1(x1)PX2(x2)⋯PXn(xn).PX1,X2,...,Xn(x1,x2,...,xn)=PX1(x1)PX2(x2)⋯PXn(xn).
If X1X1, X2X2, ..., XnXn are continuous, then they are independent if for all (x1,x2,...,xn)∈Rn(x1,x2,...,xn)∈Rn, we have
fX1,X2,...,Xn(x1,x2,...,xn)=fX1(x1)fX2(x2)⋯fXn(xn).fX1,X2,...,Xn(x1,x2,...,xn)=fX1(x1)fX2(x2)⋯fXn(xn).
If random variables X1X1, X2X2, ..., XnXn are independent, then we have
E[X1X2⋯Xn]=E[X1]E[X2]⋯E[Xn].E[X1X2⋯Xn]=E[X1]E[X2]⋯E[Xn].
In some situations we are dealing with random variables that are independent and are also identically distributed, i.e, they have the same CDFs. It is usually easier to deal with such random variables, since independence and being identically distributed often simplify the analysis. We will see examples of such analyses shortly.
Definition . Random variables X1X1, X2X2, ..., XnXn are said to be independent and identically distributed (i.i.d.) if they are independent, and they have the same marginal distributions:
FX1(x)=FX2(x)=...=FXn(x), for all x∈R.FX1(x)=FX2(x)=...=FXn(x), for all x∈R.

For example, if random variables X1X1, X2X2, ..., XnXn are i.i.d., they will have the same means and variances, so we can write
E[X1X2⋯Xn]=E[X1]E[X2]⋯E[Xn]=E[X1]E[X1]⋯E[X1]=E[X1]n.(because the Xi's are indepenednt)(because the Xi's are identically distributed)E[X1X2⋯Xn]=E[X1]E[X2]⋯E[Xn](because the Xi's are indepenednt)=E[X1]E[X1]⋯E[X1](because the Xi's are identically distributed)=E[X1]n.
